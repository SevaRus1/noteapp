# NoteApp — Мои заметки

Сайт для заметок с картинками. Сделали для учёбы.

## Кто делал:

- Матвей Еремин
- Всеволод Мягков

## Что используется:

- Flask
- Bootstrap (красивые кнопки)
- REST API (выдаёт заметки в JSON)
- Картинки (можно загружать)
- Регистрация и вход
- Поиск и фильтры по категориям

## API адреса:

 Адрес / Что делает: 

 `/api/notes` / Показывает все заметки 
 `/api/stats` / Показывает статистику 
 `/api/notes/1` / Показывает заметку с id=1

## Как запустить:

1. Открыть терминал в папке с проектом
2. Написать `pip install -r requirements.txt`
3. Написать `python app.py`
4. Открыть `https://divalc.pythonanywhere.com/`(Только с VPN)
